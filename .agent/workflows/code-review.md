# AI 编码工作流

## 任务开始时（必须执行）

// turbo
1. 读取 `ai_mistakes.md` 黑名单文件，并在第一时间的输出中回答我：已经读取`ai_mistakes.md`黑名单文件了并记住了问题，必须是这段原话
2. 在脑中复述当前任务涉及的规则（不需要输出，但要遵守）
3. 特别关注：
   - TypeScript 项目：R000, M001, M006, M007, M011
   - Python 项目：R000, M002, M005
   - 重构/删除操作：M003, M004, M010
   - 新增功能：M006, M008

## 任务进行中

- 每次代码修改前检查是否违反黑名单规则
- 如果需要使用 `any` 类型，必须说明原因
- 删除代码前必须全局搜索引用
- 修改 import 路径后必须验证所有依赖

## 任务结束时（必须执行）

1. 回顾本次对话是否出现了新的错误模式
2. 如果有新的错误类型，追加到 `ai_mistakes.md`：
   - 使用下一个编号 `M0XX`
   - 遵循现有格式：`## M0XX｜错误描述`
   - 包含：🧠 AI 行为、💥 后果、✅ 解决策略
3. 如果本次对话没有新错误，无需修改文件
